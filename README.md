
# Horse Racing

Real-time race of horses running in 8 different lanes on a straight route.


## Live

[--> Play the game <--](https://horseracingvue.web.app/)

  ## Color Reference

| Color             | Hex                                                                |
| ----------------- | ------------------------------------------------------------------ |
| Primary |  #e08a26 |
| Secondary | #00FF00 |
| Tertiary |  #6c3711 | 

## Features

- Start/Restart button
- Countdown
- Leaderbord
- Result list
- Intro
- Speaker voice-over at the beginning
